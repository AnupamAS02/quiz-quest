# Quiz Quest

A multiple-choice quiz app built with vanilla HTML, CSS, and JavaScript. No frameworks, no libraries — just the fundamentals.

## Features (planned)
- Start screen with name input
- Multiple-choice questions, one at a time
- Live score tracking
- Final results screen with percentage and feedback
- Play again without refreshing

## Tech
- HTML5
- CSS3 (Flexbox, custom properties, transitions)
- Vanilla JavaScript (ES6+)

## Run locally
Just open `index.html` in any browser.
